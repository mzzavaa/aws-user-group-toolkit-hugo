GameDay Europe Stream

A two-repo architecture for running cross-city, cross-country AWS GameDay live streams — spanning 20+ countries and 57+ User Groups. Fork the template, push your config, go live in 2 minutes.

  • Stream templates repo — all Remotion compositions + design system
  • Event repo — participant data, schedule, face photos only
  • Reusable for any city, country, or multi-region community event
Stream templates · Event repo

GameDay Event Config Repo

The event-specific config repo for AWS Community GameDay Europe. Holds only what changes per edition — participant data, schedule, face photos. Push to main and it deploys automatically via the template repo. Fork it for your own event and point it at your own template.

  • participants.ts, schedule.ts, face photos — that's it
  • Fork + set TEMPLATE_REPO variable to use your own stream design
  • Push to main → live in ~2 minutes via GitHub Actions
